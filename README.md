# Object Detection App

This is a Streamlit application for object detection using the YOLO (You Only Look Once) model. The app allows users to upload images and see detected objects along with bounding boxes and confidence scores.

## Features

- Upload an image in JPG, JPEG, or PNG format.
- Display detected objects with bounding boxes and confidence scores.
- Show a summary table of detected objects.
- Visualize precision and recall metrics (dummy data included).

## Requirements

This application requires the following Python libraries:

- `streamlit`
- `opencv-python-headless`
- `Pillow`
- `numpy`
- `pandas`
- `ultralytics`
- `matplotlib`
- `scikit-learn`
